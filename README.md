# HezarfApp 🚀

HezarfApp is a modern, premium productive tool designed to help users track their time effectively with a beautiful and intuitive interface. Built with **React**, **Vite**, and **Tailwind CSS**, it offers a seamless experience for managing tasks and visualizing performance.

## 🌟 Key Concepts & Purpose

The primary goal of HezarfApp is to provide a comprehensive **Stopwatch (Koronometre)** system that doesn't just track time, but also categorizes and visualizes it. It's designed for students, developers, and professionals who want to understand exactly where their time goes.

### Core Features:
- **Smart Stopwatch**: A precise timer with start, stop, and reset functionality.
- **Dynamic Labeling**: Categorize your time sessions (e.g., Math, Coding, Reading) with custom or suggested labels.
- **Visual Analytics**: 
    - **Pie Chart**: A clear breakdown of time distribution across different categories.
    - **Performance Radar**: A "spider" chart that visualizes your focus areas and performance patterns (requires at least 3 labels).
- **Dashboard Overview**: A condensed list of all your recorded sessions with quick-toggle "Load More" functionality for a clean UI.
- **Premium Design**: Modern aesthetics featuring smooth transitions, glassmorphism elements, and a responsive layout.

## 🛠️ Tech Stack
- **Frontend**: [React 19](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Charts**: [Recharts](https://recharts.org/)
- **Icons/SVG**: Custom inline SVG icons

## 🚀 Getting Started

### Prerequisites
- Node.js (Latest LTS version recommended)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/ahmtoz/HezarfApp.git
   ```
2. Navigate to the project directory:
   ```bash
   cd HezarfApp
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## 📊 How to Use
1. **Start the Timer**: Click the "Start" button to begin your session.
2. **Finish Session**: Click "Stop" when you're done. A modal will appear.
3. **Label Your Time**: Choose a suggested label or type your own. Pick a color to represent the category.
4. **Analyze**: View your Dashboard to see the Pie and Radar charts update in real-time.
